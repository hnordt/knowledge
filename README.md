# Knowledge

Structured study and engineering knowledge repository.

## Structure

### /books

Knowledge derived from books.

Examples:
- DDD
- Refactoring
- Pragmatic Programmer

### /docs

Knowledge derived from documentation.

Examples:
- Claude Code
- OpenAI
- Bun
- SQLite

### /concepts

Consolidated concepts that survived multiple sources and reflection.

Examples:
- Aggregates
- Event sourcing
- Deterministic execution
- Consistency boundaries

### /notes

Temporary or unconsolidated notes.

Examples:
- Hypotheses
- Observations
- Comparisons
- Scratch notes
